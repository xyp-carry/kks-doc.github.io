# KKSrobot
KKSrobot 目前支持的平台
| 平台 | 是否支持 |
| :---: | :--- |
| 微信3.9.12 | &#10004; |
| 微信4版本 | &#10004; |

## 微信3.9.12
微信3.9.12版本已有项目wxauto提供了支持，但由于开源版本已经被删库了，于是我们基于wxauto的思路，对窗口的UI进行解析，实现了对微信3.9.12版本的消息获取与回复。

### 窗口句柄监控
将群聊窗口激活(双击对话列表)，利用window API获取全部窗口句柄，再基于关键词获取窗口句柄。


### UI解析
获取窗口句柄后，利用window API获取窗口的UI元素，再基于元素的位置、大小、文本等信息，解析出消息内容，平均N秒获取一次窗口UI。


### 新消息监听
保存上一次的消息内容，与当前解析出的消息内容进行对比，根据滑动窗口判断是否有新消息。


### 消息回复
根据窗口句柄，利用windows API发送消息到指定窗口，先激活窗口为焦点窗口，发送消息后再将窗口放置最底层。

## 微信4版本 (不作为稳定版本)
微信4版本的UI无法解析，因此我们采用目标检测+OCR的方式，先对窗口截图，再利用目标检测模型识别出消息区域，最后利用OCR模型识别出消息内容。


### 新消息监听
与微信3.9.12版本的新消息监听类似，只是只以当前的界面截图进行对比，而不是解析窗口UI，平均N秒获取一次截图，对比上一次的截图，判断是否有新消息，但效果，目前没有训练识别用户ID位置的检测，因此也无法识别。
